tjz-load
========
tjz-load是一个用于在ajax加载页面时，未加载完成前的一个友好的load提示。   
使用方法：  
<script type="text/javascript" src="../src/tjz-load.js"></script>  
<div id="load"></div>  
<script type="text/javascript">  
	use('../src/tjz-load.css');  
	var load = new TjzLoad('load');  
	load.show();  
</script>
